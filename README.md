# tristenkurutz.com

personal portfolio site built with SvelteKit, TypeScript, Tailwind, and SCSS.

## stack

- [SvelteKit](https://kit.svelte.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [GT Walsheim Pro Black](https://www.gt-walsheim.com/)

## dev

```bash
npm install
npm run dev
```

## build

```bash
npm run build
```
